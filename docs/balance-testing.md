# Balance Testing

## Overview

The purpose of Balance Testing is to determine how the existing (or future) rules, game mechanics, and statistics impact the player experience.

## Metrics

Balance Testing should result in quantifiable outcomes that are used by game designers to improve the player experience. The primary opportunity area where balanace testing informs this process is in optimizing for the barriers that players may experience in the game, such as poor faction balancing / matchups and extended playtimes.

The following metrics should be calculated and evaluated for every balancing testing scenario: 
- Relative win rates of different factions
- Total estimated playtime for human players
    - Split into phases of the game (setup, placement, gameplay)
    - Estimated using the number of steps / turns
